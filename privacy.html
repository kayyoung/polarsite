Privacy Policy

Effective Date: November 18, 2025

Welcome to the Polarsite Toolkit (hereinafter "this tool"). We understand the importance of privacy to you, and this notice explains how this tool handles your data.

1. This Tool is a Pure Front-End Application

Polarsite Toolkit runs entirely in your browser. All file processing (including Markdown/TXT/HTML conversion, PDF signing, batch renaming, and image processing) is done locally. No files or content will be uploaded to any server.

2. We Do Not Collect Any User Data

This tool does not use cookies, does not record access logs, and does not embed third-party tracking code. Your privacy is 100% respected.

3. File Processing Explanation

When you upload a file or input text, the content only exists in your browser's memory. After closing the page, the data is immediately cleared and will not be stored, cached, or transmitted.

4. Third-Party Services

This tool relies on the following CDN resources (only for loading necessary JavaScript libraries for features like PDF manipulation, data parsing, and signature drawing):

  • jsPDF / html2canvas / marked (via cdnjs.cloudflare.com, cdn.jsdelivr.net)
  • pdf-lib / signature_pad (via unpkg.com, cdn.jsdelivr.net)
  • JSZip / PapaParse / XLSX (via cdnjs.cloudflare.com)

These services have their own independent privacy policies, but we only use them for functional implementation, not for data collection.

5. Contact Us

If you have any privacy-related questions, please contact us via the domain email: admin@polarsite.site
